# Datos

Estructura esperada:

```text
data/
  raw/         # capturas propias, organizadas por letra
  external/    # datasets descargados, por ejemplo LSC70 o Kaggle
  processed/   # CSV de landmarks generado para entrenamiento
```

Para la letra Ñ use la carpeta `N_TILDE`.

Ejemplo:

```text
data/raw/A/*.jpg
data/raw/B/*.jpg
data/raw/N_TILDE/*.jpg
```
